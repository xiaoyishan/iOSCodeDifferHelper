## iOSCodeDifferHelper[QQ群:594957417]

**该工具针对混淆OC开发的SDK/App**

**该工具不能混淆swift开发的SDK/App**

**该工具不能混淆unity/cocos的C++、C#代码**

## 前提

```
该工具不会开源,介意勿下载
```

## 支持运行环境:

```
MacOS
Python 3.7
```

## 支持语言

```
Objective-C (不支持Swift)
```
